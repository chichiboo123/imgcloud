# 이미지 구름 (Image Cloud)

간편하게 이미지를 업로드하고, 링크/HTML 코드를 즉시 복사해 활용할 수 있는 단일 페이지 웹앱입니다.

## 주요 기능

- 4자리 PIN 기반 잠금 화면
- 최초 접속 시 GAS(Web App) URL 등록
- 드래그 앤 드롭 / 파일 선택 / 클립보드 붙여넣기(Ctrl+V) 업로드
- 업로드 시 타임스탬프 파일명 자동 생성 (`YYYYMMDD-HHMMSS`)
- 뷰어 링크 / 직접 링크 / 썸네일 링크 제공
- HTML `<img>` 태그 코드 자동 생성 및 복사
- 구름창고(이미지 목록) 조회
- 이미지 상세 보기 및 삭제

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. PIN을 입력해 로그인합니다.
3. 첫 사용 기기라면 GAS Web App URL을 입력합니다.
4. 이미지를 업로드하고 생성된 링크를 복사해 사용합니다.

## 기술 스택

- HTML
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Lucide Icons
